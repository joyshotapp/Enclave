# 資訊安全管理SOP

**制定部門**：資訊部  
**版次**：v1.0  
**生效日期**：2026-01-01

---

        一、目的
        保護公司資訊資產，防範資料外洩、系統入侵等資安威脅。

        二、帳號管理
        1. 員工入職申請系統帳號，離職24小時內停用所有帳號
        2. 密碼複雜度：至少8碼，含大小寫、數字、特殊符號
        3. 密碼每 90 天強制更換
        4. 禁止帳號共用

        三、設備管理
        1. 公司筆電全磁碟加密（BitLocker/FileVault）
        2. 禁止個人設備存取公司機密系統
        3. 離開座位螢幕鎖定（Windows+L）
        4. 定期資安掃描與更新

        四、資料分類
        機密（紅）：薪資、健康、合約 — 嚴格存取控制
        內部（黃）：業務報告、客戶名單 — 內部限閱
        公開（綠）：官網、產品型錄 — 無限制

        五、事件通報
        發現資安事件應於 1 小時內通報資訊部，資訊部 24 小時內完成初步鑑識。

        六、違規處理
        違反資安政策依情節輕重予以警告、記過至解除職務。
